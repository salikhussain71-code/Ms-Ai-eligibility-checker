# MS AI Eligibility Checker

I built this because I am applying for MS AI in 2026 and got tired of checking 3 different university websites.
This tool takes your CGPA, IELTS, and GRE score and tells you if you are eligible for MBZUAI, KAUST, and LUMS.

This is my CS50P Week 3 project. Built with Python. No libraries.

## What It Does

1. Asks for CGPA, IELTS, GRE
2. Checks against official requirements for 3 universities
3. Shows result in a clean table with reason
4. Uses custom exception if you enter wrong CGPA or IELTS

## How to Run

```bash
python eligibility.py
